# svelte-html-repl

An HTML REPL built in svelte.

## Install

`npm i svelte-html-repl`

## Usage

```svelte
<script>
  import Repl from "svelte-html-repl";
</script>

<Repl borderColor="black" startingHTML="<em>test</em>" startingCSS="" startingJS="" tab="html" />
```

## Contributing

Use `npm run dev` to test.
